## Keyboard Shortcuts

| Shortcut           | Action                 |
|--------------------|------------------------|
| Ctrl/Cmd + K       | Copy HTML output       |
| Ctrl/Cmd + D       | Download HTML file     |
| Ctrl/Cmd + R       | Reset/Clear editor     |
